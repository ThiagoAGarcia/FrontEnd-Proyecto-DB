# FrontEnd-Proyecto-DB

**Para correr React:**

¡¡¡ Seguir en orden los siguientes pasos !!!

1. Entrar al directorio `obligatorioBDD`

   ```powershell
   cd obligatorioBDD
   ```
2. Hacer `npm i` para instalar dependencias

   ```powershell
   npm i
   ```
3. Finalmente, ejecutar npm run dev 

   ```powershell
   npm run dev
   ```
